# Hero Section + Sidebar

A simple responsive Hero Section with Sidebar built using HTML, CSS, and JavaScript.

## Features

- Responsive Layout
- Sidebar Navigation
- Hero Section
- Hover Effects
- Button Click Event
- Mobile Friendly

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Folder Structure

project/
│
├── index.html
├── style.css
└── script.js

## Run Locally

1. Download the files.
2. Open the project in VS Code.
3. Install Live Server extension.
4. Open index.html with Live Server.
